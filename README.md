# Contratos-Compra-Venda-Solidity

Dois contratos implementados em Solidity e em RCL, simbolizando uma operação de compra e venda entre duas partes, com um mediador.
A finalidade do repositório é servir de auxílio para futuras publicações sobre o tema.

- O contrato original em solidity, porém com erro é "ContratoCompraVendaErrado.sol"
- O contrato original em RCL, porém com erro é "ContratoCompraVendaErrado.rcl"
- O contrato corrigido em solidity é "ContratoCompraVendaCorreto.sol"
- O contrato corrigido em RCL é "ContratoCompraVendaCorreto.rcl"
