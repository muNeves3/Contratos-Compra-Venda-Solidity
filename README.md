# Contratos-Compra-Venda-Solidity

Dois contratos implementados em Solidity, simbolizando uma operação de compra e venda entre duas partes, com um mediador.
A finalidade do repositório é servir de auxílio para futuras publicações sobre o tema.

- O contrato original, porém com erro é "ContratoCompraVendaErrado.sol"
- O contrato corrigido é o "ContratoCompraVendaCorreto.sol"
